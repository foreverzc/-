# 正则表达式

+ 正则表达式  ==是用于匹配字符串中字符组合的模式==，在JavaScript中，正则表达式也是  对象
+ 通常用来查找、替换哪些复合正则表达式的文本，许多语言都支持正则表达式
+ 正则表达式在JavaScript 中的使用场景
  + ==例如表单验证==：用户名表单只能输入英文字母、数字或者下划线、昵称输入框中可以输入中文（匹配）
    + 比如用户名 /^[a-z0-9_]{3,16}$/
  + ==过滤掉页面内容中的一些敏感词（替换==）
  +  ==从字符串中获取我们想要的特定部分（提取）==等

## 一、语法

+ 定义规则
+ 根据规则去查找：找到则返回



+ 在JavaScript中 定义正则表达式 的语法有两种，我们先学习其中比较简单的方法：

###  简单正则表达式语法

``` js
const 变量名 = /表达式/
```

+ 其中/ / 是正则表达式的 字面量

+ ==先定义规则==

+ ==再判断是否有符合规则的字符串==：用户输入的内容

  + ==test（）方法 用来查看正则表达式与指定字符串是否匹配==

    + ``` js
      regObj.test(被检测的字符串)
      ```

    + ``` js
              const str = "我们在学习前端，希望学习前端能高新毕业"
              // 正则表达式使用：
              // 1.定义规则
              const reg= /前端/
              // 2.是否匹配
              console.log(reg.test(str));
      ```

    + 如果正则表达式与指定的字符串匹配 返回true 否则返回false

  + ==exec（）方法在一个指定字符串中执行一个 搜索匹配==

    + ``` js
      regObj.exec(被检测的字符串)
      ```

    + ``` js
              const str = "我们在学习前端，希望学习前端能高新毕业"
              // 正则表达式使用：
              // 1.定义规则
              const reg = /前端/
              // 2.是否匹配
              console.log(reg.exec(str));
      ```

    + 如果正则表达式与指定的字符串匹配 返回数组 否则为 null

  

## 二、元字符

+ 普通字符：
  + 所有的字母和数字  普通字符只能匹配字符串中与它们相同的字符
+ 元字符（特殊字符）
  + 是一些具有特殊含义的字符，可以极大提高了灵活性和强大的匹配功能
    + 比如，规定用户只能输入英文26个字母，普通字符的话 abcdef。。。。
    + 但是元字符写法：[a-z]
+ 参考文档：
  + MDN: [正则表达式 - JavaScript | MDN (mozilla.org)](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide/Regular_Expressions) 
  + 正则测试工具：http://tool.oschina.net/regex



###  元字符分类

+ ==边界符（表示位置，开头和结尾，必须用什么开头，用什么结尾）==
+ ==量词（表示重复次数==）
+ ==字符类（比如\d表示0-9）==



###  边界符

+ 正则表达式中的边界符（位置符）用来提示字符所处的位置，主要由两个字符

  | 边界符 | 说明                           |
  | ------ | ------------------------------ |
  | ^      | 表示匹配行首的文本（以谁开始） |
  | $      | 表示匹配行尾的文本（以谁结束） |

+ ^ 和$在一起 ，表示的是精确匹配 。

  + 只有一个哈才能为true
  + ==当精确匹配时 字符串只能出现/ /里面的字==

+ ``` js
          //    开头
          console.log(/^哈/.test('哈'));  //true
          console.log(/^哈/.test('二哈'));//false
          console.log(/^哈/.test('哈哈'));//true
      ```


```js
      // 结尾
      console.log(/哈$/.test('哈')); //true
      console.log(/哈$/.test('哈二'));//false
      console.log(/哈$/.test('哈哈'));//true
  
      // 一起 
      console.log(/^哈$/.test('哈'));//true
      console.log(/^哈$/.test('二哈'));//false
      console.log(/^哈$/.test('哈哈'));//false
```
###  量词（表示重复次数） 一般与精确匹配一起

+ 量词是用来  ==设定某个模式出现的次数==

  | 量词                                  | 说明             |
  | ------------------------------------- | ---------------- |
  | *     类似 >=0次                      | 重复零次或更多次 |
  | +     类似>=1                         | 重复一次或更多次 |
  | ？   类似 0\|\|1                      | 重复零次或一次   |
  | {n}    必须=== n为真                  | 重复n次          |
  | {n,}    必须>=n才为真                 | 重复n次或更多次  |
  | {n,m}   逗号两边不能有空格   >=n&&<=m | 重复n到m次       |





###  字符类

####  匹配字符集合 []

+ 后面的字符串只要包含abc 中任意一个字符，都返回true

+ 当精确匹配时  abc  只能选择其中一个 当然可以用量词 

+ ``` js
          console.log(/[abc]/.test('asdg'));    //true
          console.log(/^[abc]$/.test('asdg'));    //false
          console.log(/^[abc]$/.test('a'));       //true
          console.log(/^[abc]$/.test('b'));       //true
          console.log(/^[abc]$/.test('b'));       //true
          console.log(/^[abc]$/.test('c'));       //true
          console.log(/^[abc]{2}$/.test('ac'));   //true
  ```

####   - 连字符

+ [] 里面加上 - 连字符

+ 使用连字符 - 表示一个范围

+ ``` js
          console.log(/^[a-z]$/.test('j')); //true
  ```

+ 比如：

  + [a-z]  表示a到z26个英文字母都可以
  + [a-zA-Z]  表示大小写都可以
  + [0-9]   表示0~9的数字都可以
  + [a-zA-Z0-9]  表示大小写 数字都可以

+ 认识下

+ 腾讯qq号  ：

  + ``` js
     ^[1-9][0-9]{4,}$  (腾讯qq号从10000开始)
    ```

    + 第一位1-9
    + 第二位0-9开始 重复四次
    + =={} 只重复离自己最近的==



####  ^取反符号写在[]里面的

+ [] 里面加上^取反符
+ 比如
  + [^a-z]匹配除了小写字母以外的字符
  + 注意要写到中括号里面

####     .

+ . 匹配除换行符以外的任何单个字符



####    预定义

+ 指常见模式的简写方式

+   匹配任意的字母、数字和下划线，相当于[A-Za-z0-9_]

+ | 预定义类 | 说明                                                         |
  | -------- | ------------------------------------------------------------ |
  | \d       | 匹配0-9之间的任意数字，相当于[0-9]                           |
  | \D       | ==匹配所有0-9 以外的字符==，相当于  [^0-9]                   |
  | \w       | 匹配任意的字母、数字和下划线，相当于[A-Za-z0-9_]             |
  | \W       | ==除了所有的字母、数字和下划线以外的字符==，相当于     [^A-Za-z0-9_] |
  | \s       | 匹配空格（包括换行符、制表符、空格符等），相当于[\t\y\n\v\f] |
  | \S       | ==匹配非空格的字符==，相当于[^\t\y\n\v\f]                    |

+ ```js
  日期格式:^\d{4}-\d{1,2}-\d{1,2}
  ```







## 三、修饰符

+ 修饰符约束正则执行的某些细节，如是否区分大小写、是否支持多行匹配等
+ 语法：

``` js
/表达式/修饰符
```

+ ==i是单词ignore 的缩写，正则匹配时字母不区分大小写==
+ ==g是单词global 的缩写，匹配所有满足正则表达式的结果==

+ ==替换replace替换==

+ 语法

+ 将字符串里面的  正则表达式替换为 文本

+ ``` js
  字符串.replace(/正则表达式/，'替换的文本')
  ```

``` js
    <script>
        console.log(/^java$/.test('java')); //true
        console.log(/^java$/.test('JAVA')); //false
        console.log(/^java$/i.test('JAVA')); //true

        let str = 'java是一门编程语言，学完JAVA工资很高'
        str = str.replace(/java/ig, '前端')//不仅不区分大小写 更是全局都要替换掉
        console.log(str);
    </script>
```

``` 
        str = str.replace(/java|工资/ig, '前端')//  | 表示 或的意思
```
